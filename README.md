\### Gesture-Driven Drone Simulator



A real-time hand gesture controlled drone simulator using \*\*OpenCV\*\* and \*\*MediaPipe\*\*.



\## Features

\- Hand gesture recognition

\- Smooth gesture stabilization

\- 3D-style drone simulation

\- Radar HUD

\- Real-time webcam feed



\## Controls

| Gesture | Action |

|-------|--------|

| Fist | LAND |

| Open Palm | HOVER |

| Index + Middle | MOVE FORWARD |

| Index only | MOVE BACKWARD |

| Index + Middle + Ring | MOVE LEFT |
